# learn-java

> 🤖 AI가 가르치고, 사람이 배운다 - Spring Boot 백엔드 학습 실험

---

## 프로젝트 목표

**최종 목표**: Spring Boot 백엔드 개발 → 카톨릭 기도앱 백엔드 제작

### 학습 범위
1. **Java 기초** - 변수, 타입, 연산자, 조건문, 반복문
2. **객체지향** - 클래스, 상속, 인터페이스, 다형성
3. **실무 필수** - 예외 처리, 컬렉션, 람다, 스트림
4. **Spring Boot** - MVC, DI, JPA, Security
5. **배포** - AWS/NCP, Docker, CI/CD

---

## 학습 방식

이 프로젝트는 **AI 튜터 기반 학습 실험**입니다.

- AI가 커리큘럼에 따라 단계별로 설명
- 한 번에 하나의 항목만 학습
- 체크리스트로 진행 상황 추적
- 실행 가능한 예제 코드 제공

### 사용법
```
사용자: "오늘 어디 할 차례야"
AI: PROGRESS.md를 읽고 다음 학습 항목 안내
```
