# Java + Spring Boot 속성 코스 (경력자용)

> 목표: 2주 안에 Spring Boot로 REST API 만들 수 있는 수준
> 대상: Node.js 백엔드 경험자

---

## Week 1: Java + Spring 기초

### Day 1-2: Java 문법 (JS/TS와 다른 점만)
- [ ] 타입 시스템 (int, Integer, String, List<T>)
- [ ] 접근 제한자 (public, private, protected)
- [ ] 클래스, 상속, 인터페이스
- [ ] 제네릭 <T>
- [ ] 예외 처리 (try-catch, throws)
- [ ] 람다 + Stream API

### Day 3-4: Spring Boot 시작
- [ ] 프로젝트 생성 (start.spring.io)
- [ ] 프로젝트 구조 이해
- [ ] application.yml 설정
- [ ] @RestController, @GetMapping, @PostMapping
- [ ] @RequestBody, @PathVariable, @RequestParam

### Day 5-7: Spring 핵심 개념
- [ ] DI (의존성 주입) - @Service, @Repository, @Autowired
- [ ] 생성자 주입 패턴
- [ ] DTO 패턴
- [ ] Lombok (@Getter, @Builder, @RequiredArgsConstructor)

---

## Week 2: JPA + 실무 기능

### Day 8-9: Spring Data JPA
- [ ] Entity 클래스 (@Entity, @Id)
- [ ] Repository 인터페이스
- [ ] 기본 CRUD (save, findById, findAll, delete)
- [ ] 쿼리 메소드 (findByName 등)

### Day 10-11: 인증 + 예외 처리
- [ ] JWT 토큰 발급/검증
- [ ] Spring Security 기본 설정
- [ ] @ExceptionHandler
- [ ] @Valid 유효성 검사

### Day 12-14: 실전 패턴
- [ ] 프로젝트 폴더 구조 (controller, service, repository, dto, entity)
- [ ] @Transactional
- [ ] 간단한 CRUD API 만들어보기
